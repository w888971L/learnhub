# Designs

Early-stage thinking for the constitutional architecture and its tooling. This folder is a parking lot for the human operator's ideas — not an instruction set for agents.

Nothing here is authoritative. Nothing here should influence active development. These are sketches.

## Structure

- **`mission.md`** — the overarching vision that frames all work on this project
- **`prototypes/`** — individual design sketches, each exploring a concept that may or may not become a formal plan

## Graduation Path

A prototype becomes actionable through the existing plans framework:

1. **Prototype** (here) — unstructured thinking, no template, no review required
2. **Formal plan** (`docs/plans/`) — structured proposal using the plan template, reviewed by agents, authorized by human operator
3. **Implementation** — code changes following the approved plan
4. **Charter updates** — living docs chain keeps the Memex current

A prototype may sit here indefinitely. Not every idea needs to graduate. The value is in having a place to capture thinking without polluting the governance layer.
